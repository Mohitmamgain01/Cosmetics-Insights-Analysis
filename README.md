# 💄 Cosmetic-Insights-Analysis
**Cosmetic Insights Analysis** takes a raw, unclean cosmetics product dataset — spanning brand, product name, category, ingredients, price, popularity rank, and skin-type suitability — and turns it into a clean, interactive Tableau dashboard.

---

## 🚀 Features

- **KPI Cards** — Total Products, Total Brands, Skin Types Count
- **Brands with Average Pricing** — compare price positioning across every brand
- **Top 10 Brands by Product Count** — bar chart and pie-chart views
- **Category Level Product Distribution** — packed-bubble breakdown by category
- **Category-Wise Total Revenue** — revenue ranking across Cleanser, Eye Cream, Sun Protect, Moisturizer, Treatment, and Face Mask
- **Category-Wise Distribution for Combination Skin** — skin-type-specific category breakdown
- **Price vs Rank Scatter Plot** — relationship between price and popularity
- **Full Interactive Dashboard** — all views combined with brand/category filters and tooltips
- Responsive Flask web app with a KPI summary, embedded dashboard, and links to the GitHub repo and Tableau Public profile
