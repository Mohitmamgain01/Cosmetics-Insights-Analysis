# 💄 Cosmetics Insights - Web Integration

A Flask-based web application that embeds the **Cosmetics Insights Tableau Dashboard** and **Story** into a clean portfolio website.

## 🚀 Features

- Interactive Tableau Dashboard
- Tableau Story Integration
- KPI Summary Cards
- Responsive Design
- Direct links to Tableau Public Profile
- GitHub Repository Link

## 🛠 Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Tableau Public

## 📂 Project Structure

```
Web Integration/
│── app.py
│── templates/
│── static/
│── README.md
```

## ▶ Run Locally

```bash
pip install flask
python app.py
```

Open:

```
http://127.0.0.1:5500
```

## ⚙ Configuration

Open `app.py` and update the `CONFIG` dictionary:

- `tableau_dashboard_url`
- `tableau_story_url`
- `github_repo_url`
- `tableau_profile_url`
- `authors`
- `kpis`
